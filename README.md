1. Creacion del submodulo
`git submodule init`
2. Agregar el submodulo `git submodule add <nombre-repositorio>`
3. Adicionar los cambios `git add .`
4. Guardar los cambios `git commit -m "Mensaje"`
5. Subir los cambios `git push origin main`
